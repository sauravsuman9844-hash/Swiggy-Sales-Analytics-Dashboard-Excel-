# Swiggy-Sales-Analytics-Dashboard-Excel-
This project presents an interactive sales analytics dashboard built using Power BI, based on Swiggy food delivery data. It provides insights into sales performance across time (daily, weekly, monthly), customer ratings, order trends, and geographic distribution across India. The dashboard highlights key metrics such as total sales, average rating, and order volume, along with top-performing cities and categories. It is designed to help stakeholders quickly understand business performance and identify growth opportunities.
